## A Few Things Worth Knowing Before You Buy

**Fraud check gotcha.** ZgoCloud uses WHMCS MaxMind fraud detection. If your IP address says you're in one country but you enter a different country and phone number in the registration form, the system will likely flag your order. The information doesn't need to be "real"—it just needs to be internally consistent.

**No refunds on specials.** The Special Offer plans explicitly state no refunds. Standard plans are more flexible, but always check the Terms of Service before committing if refund policy matters to you.

**International vs China-optimized.** Plans labeled "International network" (LA Global, LA VDS, Falkenstein) are not optimized for China. If that's your use case, stick with the plans explicitly labeled "China Optimised" or "China Premium Optimised."

**IP replacement policy.** Initial IP replacement within a certain window may be free, but subsequent changes cost around $6-10 per IP depending on type. If you need a clean IP for a specific service, verify the IP's reputation immediately after provisioning.

**Support channels.** 24/7 support via ticket system and a Telegram channel for community discussions. No live chat, so factor in ticket response time for urgent issues.
